O SkyLife RP

Komunita

Přátelská komunita hráčů, kteří si užívají roleplay a společně vytvářejí jedinečný herní svět.

Vlastní Scripty

Náš server nabízí unikátní herní mechaniky a skripty, které nenajdete nikde jinde.

Pravidelné Aktualizace

Neustále vylepšujeme server a přidáváme nový obsah, aby hra zůstala vždy svěží a zábavná.

SkyLife RP je český roleplay server zaměřený na realistický herní zážitek. Náš server nabízí propracovaný ekonomický systém, různorodé pracovní příležitosti, vlastní vozidla a mnoho dalšího. Připojte se k nám a staňte se součástí rostoucí komunity hráčů!
